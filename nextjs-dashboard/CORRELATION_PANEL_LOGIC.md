**데이터 소스:** 정적 데이터 (`lib/correlationData.ts`)

**참고:** 현재는 사전 분석된 상관관계 데이터를 사용하며, 향후 Supabase에서 실시간 계산 가능

## 🔢 상관계수 계산 방식

### 상관계수 (Correlation Coefficient)

**범위:** -1 ~ +1

- **+1**: 완벽한 양의 상관관계 (한쪽 증가 시 다른 쪽도 증가)
- **0**: 상관관계 없음
- **-1**: 완벽한 음의 상관관계 (한쪽 증가 시 다른 쪽은 감소)

### 통계적 유의도 (p-value)

- **p < 0.001**: 매우 유의미 (***)
- **p < 0.01**: 유의미 (**)
- **p < 0.05**: 약간 유의미 (*)
- **p >= 0.05**: 유의미하지 않음

### 유의도 분류

```typescript
significance: 'high' | 'medium' | 'low'
```

- **high**: p-value < 0.05, 상관계수 절댓값 > 0.04
- **medium**: p-value < 0.1, 상관계수 절댓값 > 0.02
- **low**: 그 외

## 🎨 표시 로직

### 1. 퍼센트 변환

```typescript
getCorrelationPercentage(correlation: number): number {
  return Math.abs(correlation) * 100;
}
```

**예시:**
- 상관계수 0.133 → **13.3%**
- 상관계수 0.090 → **9.0%**
- 상관계수 -0.008 → **0.8%** (절댓값 사용)

### 2. 색상 동그라미 (유의도 표시)

```typescript
getSignificanceColor(significance: 'high' | 'medium' | 'low'): string {
  switch (significance) {
    case 'high': return '#22c55e';    // 초록색
    case 'medium': return '#eab308';  // 노란색
    case 'low': return '#6b7280';     // 회색
  }
}
```

**표시:**
- 🟢 **초록색**: 유의미한 상관관계 (p < 0.05)
- 🟡 **노란색**: 약한 상관관계 (p < 0.1)
- ⚪ **회색**: 관계 없음 (p >= 0.1)

### 3. 진행 바 색상

```typescript
getBarColor(correlation: number, significance: string): string {
  if (significance === 'low') return '#4b5563';  // 회색
  return correlation > 0 ? '#22c55e' : '#ef4444'; // 초록/빨강
}
```

**표시:**
- **초록색**: 양의 상관관계 (correlation > 0)
- **빨간색**: 음의 상관관계 (correlation < 0)
- **회색**: 유의도 낮음 (significance === 'low')

### 4. 진행 바 너비

```typescript
width: `${percentage}%`
```

**예시:**
- 상관계수 0.133 → 13.3% 너비
- 상관계수 0.090 → 9.0% 너비
- 상관계수 0.72 → 72% 너비

### 5. 부호 표시

```typescript
const sign = correlation >= 0 ? '+' : '';
```

**표시:**
- 양수: **+13.3%**
- 음수: **-0.8%** (부호 자동 표시)

## 📊 데이터 구조

### CorrelationItem 인터페이스

```typescript
interface CorrelationItem {
  label: string;                    // 항목 이름
  correlation: number;               // 상관계수 (-1 ~ 1)
  pValue?: number;                  // p-value (통계적 유의도)
  significance: 'high' | 'medium' | 'low';  // 유의도 분류
  description?: string;              // 간단한 설명
  detailedAnalysis?: string;         // 상세 분석 (툴팁용)
}
```

### 예시 데이터

```typescript
{
  label: '고래 거래 빈도 (ETH)',
  correlation: 0.133,
  pValue: 0.000000,
  significance: 'high',
  description: '고래 거래가 많을수록 ETH 가격 상승',
  detailedAnalysis: `📊 가장 강력한 상관관계 (r=0.133, p<0.001)
  
• 고래 거래 빈도가 1 표준편차 증가 시 ETH 가격 평균 13.3% 상승
• 7,992개 데이터 포인트 분석 결과
• 통계적으로 매우 유의미 (p-value < 0.000001)
• ETH가 BTC보다 고래 거래에 더 민감하게 반응

💡 활용: 고래 거래 급증 관찰 시 ETH 매수 타이밍으로 활용 가능`
```

## 🔍 상관관계 해석 가이드

### 상관계수 강도

| 상관계수 절댓값 | 강도 | 해석 |
|---------------|------|------|
| 0.7 ~ 1.0 | 매우 강함 | 강한 선형 관계 |
| 0.4 ~ 0.7 | 강함 | 명확한 관계 |
| 0.2 ~ 0.4 | 중간 | 약한 관계 |
| 0.0 ~ 0.2 | 약함 | 거의 관계 없음 |

### 실제 예시

1. **뉴스 수 (급증 이벤트)**: 0.72 → 매우 강한 상관관계
   - 뉴스 급증 시 시장 활동이 12.7배 증가

2. **고래 거래 빈도 (ETH)**: 0.133 → 약한 상관관계
   - 하지만 통계적으로 매우 유의미 (p < 0.001)
   - 실용적으로 활용 가능

3. **텔레그램 감정**: -0.008 → 거의 관계 없음
   - p-value 0.49로 통계적으로 유의미하지 않음

## ⚠️ 주의사항

1. **정적 데이터**
   - 현재는 사전 분석된 데이터 사용
   - 실시간 계산이 아닌 고정된 값

2. **상관관계 ≠ 인과관계**
   - 상관관계가 있다고 해서 반드시 인과관계는 아님
   - 다른 요인이 영향을 줄 수 있음

3. **과거 데이터 기반**
   - 분석은 과거 데이터를 기반으로 함
   - 미래 성과를 보장하지 않음
