# 📊 통계 카드 계산 로직

## 🎯 개요

차트 하단에 표시되는 **통계 카드들**은 현재 차트에 표시된 데이터의 **평균값**을 계산하여 보여줍니다.

**데이터 소스:** 차트에 표시된 `data` 배열 (필터링된 데이터)

---

## 📈 통계 카드 종류

### 1️⃣ 평균 고래 거래 (항상 표시)

**계산 공식:**
```typescript
평균 = sum(whale_tx_count) / 데이터_개수
```

**코드:**
```typescript
(data.reduce((sum, d) => sum + d.whale_tx_count, 0) / data.length).toFixed(0)
```

**표시:**
- 값: 정수로 반올림 (소수점 없음)
- 단위: "건"
- 색상: 보라색

**예시:**
- 데이터: [50, 60, 40, 30, 35]
- 평균: (50 + 60 + 40 + 30 + 35) / 5 = **43건**

---

### 2️⃣ BTC 평균 가격 (BTC 또는 Both 선택 시 표시)

**계산 공식:**
```typescript
평균 = sum(btc_close) / 데이터_개수 / 1000
```

**코드:**
```typescript
(data.reduce((sum, d) => sum + d.btc_close, 0) / data.length / 1000).toFixed(1)
```

**표시:**
- 값: 소수점 1자리 (천 단위로 변환)
- 단위: "$" + "K" (예: $98.2K = $98,200)
- 색상: 주황색

**예시:**
- 데이터: [98200, 98500, 98000, 97500, 97800]
- 평균: (98200 + 98500 + 98000 + 97500 + 97800) / 5 / 1000 = **$98.2K**

---

### 3️⃣ BTC 평균 변화 (BTC 또는 Both 선택 시 표시)

**계산 공식:**
```typescript
평균 = sum(btc_change) / 데이터_개수
```

**코드:**
```typescript
const avgChange = data.reduce((sum, d) => sum + d.btc_change, 0) / data.length;
// 양수면 초록색, 음수면 빨간색
avgChange > 0 ? 'text-green-400' : 'text-red-400'
// 양수면 '+' 기호 추가
avgChange > 0 ? '+' : ''
avgChange.toFixed(2)
```

**표시:**
- 값: 소수점 2자리
- 단위: "%"
- 색상: 
  - 양수: 초록색 (text-green-400)
  - 음수: 빨간색 (text-red-400)
- 기호: 양수일 때 "+" 표시

**예시:**
- 데이터: [+0.5, -0.2, +0.1, -0.3, -0.1]
- 평균: (0.5 - 0.2 + 0.1 - 0.3 - 0.1) / 5 = **-0.01%** (빨간색)

---

## 🔍 계산 특징

### 1. 필터링된 데이터 사용

통계 카드는 **현재 차트에 표시된 데이터**만 사용합니다:

- 시간 범위 필터 (7일, 30일, 90일)
- 코인 선택 필터 (BTC, ETH, Both)
- 날짜 선택 필터
- 드래그로 이동한 범위

**예시:**
- 전체 데이터: 30일치 (720개 포인트)
- 선택된 범위: 최근 7일 (168개 포인트)
- **계산 대상**: 168개 포인트만 사용

### 2. 조건부 표시

**평균 고래 거래:**
- 항상 표시 (코인 선택과 무관)

**BTC 관련 카드:**
- `selectedCoin !== 'eth'` 일 때만 표시
- 즉, BTC 또는 Both 선택 시 표시

**ETH 관련 카드:**
- `selectedCoin !== 'btc'` 일 때만 표시
- 즉, ETH 또는 Both 선택 시 표시

### 3. 반올림 처리

| 카드 | 반올림 방식 | 예시 |
|------|------------|------|
| 평균 고래 거래 | 정수 (0자리) | 43건 |
| BTC 평균 가격 | 소수점 1자리 | $98.2K |
| BTC 평균 변화 | 소수점 2자리 | -0.01% |
| ETH 평균 가격 | 정수 (0자리) | $3264 |
| ETH 평균 변화 | 소수점 2자리 | -0.01% |

## 📝 계산 예시

### 전체 예시

**데이터 (5개 포인트):**
```typescript
[
  { whale_tx_count: 50, btc_close: 98200, btc_change: 0.5, eth_close: 3264, eth_change: 0.3 },
  { whale_tx_count: 60, btc_close: 98500, btc_change: -0.2, eth_close: 3280, eth_change: -0.2 },
  { whale_tx_count: 40, btc_close: 98000, btc_change: 0.1, eth_close: 3250, eth_change: 0.1 },
  { whale_tx_count: 30, btc_close: 97500, btc_change: -0.3, eth_close: 3240, eth_change: -0.4 },
  { whale_tx_count: 35, btc_close: 97800, btc_change: -0.1, eth_close: 3286, eth_change: -0.1 }
]
```

**계산 결과:**

1. **평균 고래 거래**
   - (50 + 60 + 40 + 30 + 35) / 5 = **43건**

2. **BTC 평균 가격**
   - (98200 + 98500 + 98000 + 97500 + 97800) / 5 / 1000 = **$98.0K**

3. **BTC 평균 변화**
   - (0.5 - 0.2 + 0.1 - 0.3 - 0.1) / 5 = **-0.01%** (빨간색)

4. **ETH 평균 가격**
   - (3264 + 3280 + 3250 + 3240 + 3286) / 5 = **$3264**

5. **ETH 평균 변화**
   - (0.3 - 0.2 + 0.1 - 0.4 - 0.1) / 5 = **-0.01%** (빨간색)

```
    └─ 통계 카드 계산 (필터링된 데이터 사용)
        ↓

```

## ⚠️ 주의사항

1. **빈 데이터 처리**
   - `data.length === 0`이면 `NaN` 또는 `Infinity` 발생 가능
   - 실제 코드에서는 데이터가 있을 때만 표시됨

2. **필터링 영향**
   - 통계는 현재 화면에 표시된 데이터만 반영
   - 시간 범위나 코인 선택 변경 시 자동 재계산

3. **정확도**
   - BTC 가격은 천 단위로 변환하여 표시 (소수점 1자리)
   - 변화율은 소수점 2자리까지 표시
